# Decision Tree .arff

- Python program to read any “.arff” file and then display a graphical interface of the tree

- **Libraries**
  - matplotlib
  - pandas
  - scipy
  - sklearn

### Setup
```bash
pip install matplotlib
```
```bash
pip install pandas
```
```bash
pip install scipy
```
```bash
pip install scikit-learn
```

```bash
git clone https://github.com/DevJpLg/Crypto-Tracker.git

cd Crypto-Tracker

python main.py
```
