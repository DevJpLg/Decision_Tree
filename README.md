# Decision Tree .arff

- Python program to read any “.arff” file and then display a graphical interface of the tree

- **Libraries**
  - matplotlib
  - pandas
  - scipy
  - sklearn

### Setup
`pip install matplotlib`

`pip install pandas`

`pip install scipy`

`pip install scikit-learn`

```bash
git clone https://github.com/DevJpLg/Decision_Tree.git

cd Decision_Tree

python main.py
```
